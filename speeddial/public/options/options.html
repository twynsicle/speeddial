Saving pages allows you to save pages by clicking the floppy disc in the url bar. Opening a site from the speed dial will then take you to the last save page.

This should be useful to save navigating to test products, or to easily return to a particular area of the site you are working on.

Note: this overrides the landing page setting when a saved page is available.
